Anakart

Anakartlar, bilgisayarın parçalarını üzerinde barındıran tümleşik, elektronik devre kartlarıdır. Anakartların üzerinde işlemci yuvası, RAM yuvası gibi yuvalar bulunur.

RAM

RAM(Random Access Memory = Rasgele Erişimsel Bellek) verilen üzerinde geçici olarak tutulduğu, elektrik kesildiğinde üzerindeki verilen silindiği elemandır.
